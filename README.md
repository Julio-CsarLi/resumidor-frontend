Los pasos para la ejecucion de la pagina web son los siguientes:
1.- Dar click en el enlace de la pagina
2.- Se le solicitara al usuario que se registre
3.- Una vez registrado se le pedira que inicie sesion
4.- Se encontrara con un cuadro de texto donde el usuario pondra su texto
5.- Una vez ingresado el texto se dara clic en el boton generar resumen
6.- En el cuadro de texto de abajo se generara su resumen del texto ingresado
